# eletronics and arduino Learning

This repository documents my journey learning eletronics and arduino.

## Objectives

- to develop a solid foundation in how to do what I need for my projects.
- Learn how to assemble and verify the circuits and projects that I will create.
- Build the necessary foundation for embedded systems and engineering/robotics projects.

## Content

- **Fundamentals of Electronics:** Electrical quantities (Voltage, Current, Resistance), Ohm's Law and reading schematics.
- **Basic Components:** Use of resistors, LEDs, diodes, capacitors and breadboard.
- **C/C++ Programming for Arduino:** Control structures (`if/else`, `loops`), variables, functions and manipulation of digital/analog ports.
- **Sensors and Actuators:** Integration of sensors (presence, temperature, luminosity) and control of motors (servomotors and DC motors).
- **Communication and Protocols:** Communication protocols (I2C, SPI, Serial) and data display on displays (LCD/OLED). - [ ] **Integrative Projects:** Develop projects that use everything I've learned up to the point I'm at (IoT).
